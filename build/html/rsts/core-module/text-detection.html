文本检测¶

随着多媒体以及互联网技术的快速发展，人们获取图片的方式越来越多，怎样从丰富的图片中摘取自己需要的信息就变得越来越重要。而随着人工智能和深度学习技术的发展，高效读取自然场景中文本的应用需求也急剧增加，自然场景文本定位和识别的应用场景也越来越多，如扫描版财务合同的识别、翻译机上的拍照翻译功能、各种证件识别等等。

环境搭建¶

环境搭建需要以下几个步骤：

  • 下载安装环境依赖包， GPU/CPU下载

  • 解压依赖包: tar -xvf build.tar.gz

  • 进入build目录： cd build

  • 进行安装： bash install.sh /path/to/conda/installation/dir

注解

安装路径/path/to/conda/installation/dir开始不存在，执行安装命令后会自动创建安装目录。进入安装目录初始化conda，重新登录账号即可。
