环境搭建¶

环境搭建需要以下几个步骤：

  • 下载安装环境依赖包， GPU/CPU下载

  • 解压依赖包: tar -xvf build.tar.gz

  • 进入build目录： cd build

  • 进行安装： bash install.sh /path/to/conda/installation/dir

注解

安装路径/path/to/conda/installation/dir开始不存在，执行安装命令后会自动创建安装目录。进入安装目录初始化conda，重新登录账号即可。
